
### Command: `ai-services version`

**Description:**  
Displays detailed version information 

**Usage**

```bash
ai-services version
ai-services [command] --version
ai-services [command] -v
```

```bash
% ai-services version
Version: 732d48a
GitCommit: 732d48a
BuildDate: 2025-11-04T07:33:42Z
% ai-services --version
ai-services version 732d48a
% ai-services -v
ai-services version 732d48a
```
